# Roadmap

## v1.x

- Harden exporter API.
- Improve test coverage.
- Add release ZIP automation.
- Add documentation examples.
- Validate live WordPress migration scenarios.

## v2.x

- WooCommerce export.
- Menus.
- Comments.
- Redirects.
- Custom post types.
- ACF and custom fields.
- WPML and Polylang support.

## Long-term

WP 2 TNCMS should remain an export-only data provider. Import behavior belongs in TNCMS Importer.
