# Code of Conduct

We want the TNCMS community to be respectful, useful and welcoming.

## Expected Behavior

- Be respectful.
- Be constructive.
- Assume good intent.
- Help others learn.

## Unacceptable Behavior

- Harassment.
- Personal attacks.
- Public sharing of private information.
- Disruptive or abusive behavior.

Maintainers may remove comments, issues or pull requests that violate this code.
