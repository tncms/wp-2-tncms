# Contributing

Thank you for helping improve WP 2 TNCMS.

## Workflow

1. Fork the repository.
2. Create a feature branch.
3. Keep changes focused.
4. Follow WordPress Coding Standards.
5. Add or update tests where possible.
6. Submit a pull request.

## Development Commands

```bash
composer install
composer lint
composer analyse
```

## Compatibility

The REST API v1 is treated as a stable public contract. Do not remove or rename existing response fields without a new API version.
